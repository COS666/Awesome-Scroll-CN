交易（Swap）

单击Swap导航栏中的按钮以打开 Swap 应用程序。

Swap 应用程序是 Uniswap V2 的分叉。交换代币和提供流动性的步骤与 Uniswap 几乎相同。为了方便起见，我们在这里为这两个操作提供一个简单的指南。
