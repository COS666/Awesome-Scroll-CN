代币转账

代币转账

您可以直接使用您的钱包在 Scroll L1 或 Scroll L2 网络内转移代币。

  1. 打开您的钱包并切换到Scroll L1或Scroll L2网络。
  2. 单击中间的发送按钮，然后在文本框中键入要转移到的地址。
  3. 在“资产”框中选择代币（TSETH或TSUSDC），然后输入您要转账的代币数量。
  4. 单击 下一步 按钮，然后单击确认 按钮发送交易。
  5. 发送后，您可以在钱包的 活动 选项卡中找到交易。
